## botc-icons

Role icons can be downloaded for local use from [tomozbot/botc-icons](https://github.com/tomozbot/botc-icons) by running:

```bash
npm run assets:icons
```

The downloaded icon files are written to `public/clocktower-assets/role_icon/` and are gitignored. They are a local runtime cache, not part of this project's MIT-licensed source distribution.

The botc-icons README describes the repository as a collection of playtest character art for use in unofficial tools and notes that icon copyrights remain with their respective owners.
